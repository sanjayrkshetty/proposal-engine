# Scope Estimate — Gulf Digital Holdings
**Service:** Web Application Penetration Test — Enterprise Tier

## Effort Estimate
| Phase | Days |
|-------|------|
| On-site | 2 |
| Remote | 10 |
| Reporting | 3 |
| **Total** | **15** |

**Indicative Price:** ₹150,000 – ₹250,000 INR

## In Scope
- Web application penetration testing of Gulf Digital Holdings' customer-facing web applications
- Testing of web application APIs and interfaces
- Assessment of security controls for financial data
- Evaluation of compliance with PCI DSS requirements
- Testing of web application authentication and authorization mechanisms
- Identification of vulnerabilities and weaknesses in web application code
- Assessment of web application configuration and patch management

## Out of Scope
- Network penetration testing
- Social engineering attacks
- Physical security assessments

## Deliverables
- Comprehensive web application penetration test report
- Executive summary and presentation of findings
- List of identified vulnerabilities and recommended remediations
- PCI DSS compliance assessment report

## Assumptions
- Client will provide necessary access and information in a timely manner
- Web applications are built using standard technologies and frameworks
- No significant changes to web applications or infrastructure during the engagement

## SLA Commitments
- Initial response: 4 hours
- Containment: 24 hours
- Report delivery: 5 business days

## Commercial Risks
⚠ Scope creep due to discovery of additional web applications or APIs
⚠ Delays in receiving necessary information or access from the client
⚠ Changes in PCI DSS requirements or regulations during the engagement
